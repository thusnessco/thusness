# TOKEN_ADDENDUM

No new tokens are introduced for this rollout. Use `design_handoff_thusness/tokens.css` as-is.

The full set in play is:

| Token | Value | Usage |
|---|---|---|
| `--thusness-bg` | `#efece1` | Page background, warm cream |
| `--thusness-ink` | `#1a1915` | Primary text, headings |
| `--thusness-ink-soft` | `#3d3a2f` | Body prose |
| `--thusness-muted` | `#8a8672` | Secondary labels, dates, captions, kickers |
| `--thusness-rule` | `#c7c2b0` | Hairline rules, card borders |
| `--thusness-red` | `#c23a2a` | Signature dot — once per page only |

If a future surface genuinely needs a new color or size, add it here as a row with `name | value | usage` and document where it gets used. Do not introduce ad-hoc colors or sizes inside components.
